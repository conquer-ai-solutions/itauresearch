Data Source Inventory

Bloomberg
Market Data
Format: CSV, Parquet
Access: DL REST API, SFTP, BLPAPI, Excel BDH/BDP/BDS
Refresh: Real-time to daily
License: Per-seat or enterprise DL. AI/ML use requires amendment.
Capital IQ
Fundamentals
Format: JSON (REST), pipe-delimited flat files
Access: CIQ Pro API (REST/HTTPS), Xpressfeed (SFTP bulk), Excel plugin
Refresh: Daily bulk, on-demand API
License: Enterprise. Distribution restricted without authorization.
Economatica
LatAm Equities
Format: CSV/TXT
Access: REST API feed, Excel plugin, Terminal
Refresh: Scheduled/on-demand
License: Commercial. Direct exchange feeds, 100% Brazil coverage.
CVM Filings
Regulatory
Format: XML, CSV
Access: Bulk download (dados.cvm.gov.br)
Refresh: As filed (ITR quarterly, DFP annual)
License: Free public access.
IBGE SIDRA
Economic Stats
Format: JSON, CSV
Access: REST API (servicodados.ibge.gov.br/api/v3)
Refresh: Varies by series
License: Free public API.
CONAB/ANP
Commodities
Format: PDF (CONAB), CSV (ANP)
Access: Bulk download, PDF extraction pipeline
Refresh: Monthly forecasts/production
License: Free public access.
Salesforce CRM
Internal
Format: JSON (REST API)
Access: Salesforce API, internal doc stores
Refresh: Continuous
License: Internal — no external licensing.
Web Search
News & IR
Format: Unstructured HTML
Access: Agentic web search with extraction
Refresh: Real-time
License: Public domain.

The Private Company Research Gap

The Problem
• Bloomberg/CIQ don't cover private companies
• Brazilian sources are fragmented & unstructured
• Sector-specific data spread across multiple free/commercial feeds
• Manual cross-referencing is time-intensive
The Solution
• IBGE SIDRA API for macro/sector baseline
• CVM filings for public comparables
• CONAB/ANP for commodity-specific thesis
• Agentic web search for news & IR pages
• Smart data fusion in context window

Option B: Analyst-in-the-Loop

Data Ingestion Layer
Analysts export from Bloomberg Excel (BDH/BDP) → CSV upload
Analysts export from Capital IQ plugin (formulas) → CSV upload
Analysts export from Economatica terminal → CSV upload
Public sources (CVM, IBGE, ANP) → still fully automated
CONAB → automated PDF extraction
Internal data → manual upload or Salesforce API
Research Engine
AI ingests uploaded analyst files + automated public feeds
Analyst retains full control over licensed data quality
⏱ Timeline: 5–6 weeks
✓ No new licensing required. Works within existing analyst workflow.

Option C: Hybrid (Recommended)

Tier 1 (Internal)
Automated via Salesforce API — no licensing barrier
Tier 2 (Licensed)
Analyst-in-the-loop for Bloomberg/CIQ/Economatica exports — respects licensing terms
Tier 3 (Public)
Fully automated: CVM REST + IBGE SIDRA API + ANP bulk download + CONAB PDF extraction + agentic web search
Why Recommended
Start proving value immediately with Tiers 1 & 3 (no licensing negotiation). Layer in Tier 2 via analyst exports. Negotiate API access in parallel for future automation.
⏱ Timeline: 6–8 weeks
